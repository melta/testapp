Concordance test application.
=============================

This is simple application that generates an alphabetical list of all word occurrences,
labeled with word frequencies.

Installation
------------

Before installation please make sure that you have 'Git' and 'Vagrant' installed on your host machine.
Also make sure that port:9999 is not in use on your host machine.

To install application simply run install.sh script:
$ ./install.sh

This scripts will do the following:
1. Clone application repository
2. Set up virtual machine with application using vagrant
3. Run default browser with application url

P.S if browser will not start automatically, please, run your browser and open the following url:
http://localhost:9999/words/
